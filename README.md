# 2. GAN for Anime-style Image Generation

## Overview
This project uses **Generative Adversarial Networks (GANs)** to transform real human images into anime-style images. It focuses on **image-to-image translation** and realistic style generation.

## Problem Statement
Creating high-quality stylized images is challenging using traditional methods. This project leverages GANs to generate **visually appealing anime-style images** from real-world inputs.

## Approach
- Implemented Generator and Discriminator networks
- Used adversarial training to improve image quality
- Preprocessed dataset for better feature learning
- Applied image-to-image translation techniques

## Tech Stack
- Python
- TensorFlow / PyTorch
- OpenCV
- NumPy

## Results
- Generated realistic anime-style images
- Learned style transformation effectively
- Improved visual quality over training iterations
